# 家計在庫アプリ（Web版）

Excel 版で作っていた **買い物入力 → 在庫自動反映 → 買い物リスト** の流れを、**GitHub Pages** 上で動く静的 Web アプリとして実装しました。

## 機能
- 最初のタブ **「買い物入力」** で **日付 / 商品名 / 区分（購入・使用） / 数量 / 備考** を入力 → **履歴に追記**
- 在庫は **初期在庫 + 購入合計 − 使用合計** を自動計算。**未登録品は自動で在庫に追加**（スピル相当の動的追加）。
- **発注点** を下回った品目は「⚠ 発注必要」。**買い物リスト** は提案数量（MAX(1, 発注点−現在庫)）付きで自動生成。
- データは **localStorage** に保存。**JSON のバックアップ/復元**に対応。

## デプロイ手順（GitHub Pages）
1. このリポジトリを作成し、以下のファイルを配置：
   - `index.html`, `styles.css`, `app.js`
2. GitHub の **Settings → Pages** で Branch を `main` / folder `/(root)` に設定し **Save**。
3. 数十秒後、Pages の URL（`https://<your-account>.github.io/<repo>/`）でアクセスできます。

> ※ サーバー不要のクライアントサイド SPA です。データは端末ごとに保持されます。複数端末で同期したい場合は、JSON をエクスポートして別端末でインポートしてください。
